# **3.1 判断数据类型**

使用反射，引入反射包

reflect.TypeOf(x)

实例：



```
package main

import (
    "fmt"
    "reflect"
)

func main() {
    var x string = "aaa"
    fmt.Println("type:", reflect.TypeOf(x))
}
```
